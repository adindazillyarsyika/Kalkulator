RESTful pada aplikasi Kalkulator

NAME PATH HTTP VERB PURPOSE MONGOOSE METHOD
INDEX ROUTES
Landing Pages / GET Menampilkan landing page atau index pages N/A
Login Form /login GET Menampilkan halaman Form Login N/A
handle login /login POST menghandle login pengguna N/A
Register Form /register GET menampilkan halaman register N/A
handle Register /register POST menghandle register N/A
CALCULATOR ROUTES
Calculators /calculators GET menampilkan semua aplikasi Kalkulator Calculator find()
per Calculator /calculators/id GET menampilkan per aplikasi Kalkulator Calculator findById()
Bookmark Calculator /calculators/id/bookmark PUT bookmark Kalkulator yang dipilih lalu redirect kembali ke halaman User findByIdAndUpdate()
USER ROUTES
USER /user GET menampilkan semua informasi tentang pengguna dan lain - lain User findById()
EDIT USER /user/edit GET Menampilkan form edit data pengguna User findById()
UPDATE USER /user PUT UPDATE informasi pengguna lalu redirect ke halaman pengguna User findByIdAndUpdate()
